# 🤖 Telegram AI Agent — Anantyug.AI

An intelligent AI-powered Telegram chatbot built using **n8n**, **OpenAI GPT-4o Mini**, and **Telegram Bot API**.
This workflow creates a smart conversational assistant capable of replying to users in real-time with memory support and automation capabilities.

---

## 🚀 Overview

This project is a fully automated Telegram AI Agent designed to:

* Receive user messages from Telegram
* Process conversations using OpenAI
* Maintain conversational memory
* Generate intelligent responses
* Send replies automatically back to Telegram

Built under the vision of **Anantyug.AI** — focused on creating powerful AI automations and agents.

## 🔧 Setup Guide

### 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/telegram-ai-agent.git
cd telegram-ai-agent
```

---

### 2️⃣ Setup Telegram Bot

* Open Telegram
* Search for `@BotFather`
* Create a new bot
* Copy your Bot Token

---

### 3️⃣ Configure n8n

Import the workflow JSON into n8n.

Add credentials for:

* Telegram API
* OpenAI API

---

### 4️⃣ Add OpenAI Model

Recommended Model:

```text
gpt-4o-mini
```

---

### 5️⃣ Activate Workflow

Click:

```text
Execute Workflow → Activate
```

Now your Telegram AI Bot is live 🚀
